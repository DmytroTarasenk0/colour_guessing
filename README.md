# Colour Guessing Game

A fully functional Colour Guessing game built with **React**. Features a simple layout and logic since it's a fast-build project.

## Live Demo

Play the game here: https://colour-guessing-livid.vercel.app/

## Features

- **Interactive Gameplay:** Observe the target colour box and try to recreate the exact shade using an interactive colour-palette box.
- **Dynamic Generation:** Colours are randomly generated every round for endless replayability.
- **Responsive Design:** Fully playable on both desktop and mobile devices.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/DmytroTarasenk0/colour_guessing.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the dev server:
    ```bash
    npm run dev
    ```
